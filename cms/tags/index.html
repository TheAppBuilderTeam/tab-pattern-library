Tags are used to show what user groups can access a particular piece of content and are found within the “audience” tab. Each user group is shown as a tag and app editors can remove user groups from the audience by clicking on the “x” on the tag. If a user group is added to the audience of a piece of content but the users have no way to access it because they are not able to view the parent list, the tag will appear in red.

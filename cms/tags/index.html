Tags are used in two cirumstances; to show what user groups can access a particular piece of content and to set "watchers" to content.

Audience Tags

A piece of content published into the app can be made to only be viewable by some users or user groups. The App Editor can set these user groups in the “audience” tab of a piece of content.

Defining the audience of a piece of content

Each user group is shown as a tag and app editors can remove user groups from the audience by clicking on the “x” on the tag. If a user group is added to the audience of a piece of content but the users have no way to access it because they are not able to view the parent list, the tag will appear in red.

  • Group that can see this content
  • Another group name
  • Group
  • Group with no way to access content

Watchers

In the Editor section of the CMS the App Editor can choose to add users as watchers on individual pieces of content. These watchers will be notified about social activity on the piece of content. Choosing users as watchers works similarly to adding user groups to the audience of a piece of content.

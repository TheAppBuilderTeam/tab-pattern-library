Throughout the Content Management System, media modals are used for inserting, organising and uploading media. These modals include the existing modal and the media library modal.

Media Library Modal

The Media Library modal appears in a few instances throughout the CMS, it is mainly used for inserting media or choosing featured images on lists items and modular page items.

App editors can upload images, files and videos to the media library via the modal. This can be done by using the drag and drop option or they can selected the 'Choose File' link to add local files.

Existing Modal

The Existing Modal is used when an app editor is adding an existing item to a list or modular page. It features filters and a sort by option, as well as search. The filter and sort by are always prefiltered to 'newest' and 'all content'. See below for examples of this.

Existing Modal on a modular page

The existing modal has the same layout and design as the media library modal, although it does not have a selection panel on the right.

The existing modal

When there is no existing content yet the 'No exisiting content' message and existing icon will show. When there are no results, the 'No results found' message and spy glass are shown (see below).

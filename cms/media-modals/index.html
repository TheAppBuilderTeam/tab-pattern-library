Throughout the Content Management System, media modals are used for inserting, organising and uploading media. These modals include the existing modal and the media library modal.

Media Library Modal

The Media Library modal appears in a few instances throughout the CMS, it is mainly used for inserting media or choosing featured images on lists items and modular page items.

App editors can upload images, files and videos to the media library via the modal. This can be done by using the drag and drop option or they can selected the 'Choose File' link to add local files.

Links

Links are used instead of buttons for many other actions a user can take on a screen. For example, adding new users to channels or user groups. These blue links will often had a “+” in front of the text to indicate the adding action.

Micro links are also used in the Editor section of the CMS to display the number of groups that can view content and link to the audience tab for that content.

Upload Buttons

Upload buttons are used when adding an image, file or video to a modular page. They are present inside the image, file and video widgets before any content has been added. The widget control buttons are displayed beside the upload widget.

Each button features an upload icon and link text underneath as label. The media library button is presented beside the upload buttons as an option too.
